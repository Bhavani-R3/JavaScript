Proxy Constructor

new Proxy(target,handler)

target => object / function / url which we want to proxy

handler() => callback function

A Proxy in js can be used to modify the fundamental operation of the an object. A Proxy on an object is itself an object, that has traps.

Traps may be triggered when operations are performed on the proxy. (property lookup [get], function calling, modifing properties, adding properties [set], removing properties[delete])
